# Three.js Journey — Self Learning

Personal learning repository following [Bruno Simon's Three.js Journey](https://threejs-journey.com/) course.  
Each folder is a self-contained exercise covering a specific lesson or concept.

---

## About
CS student exploring creative and graphics programming. Exercises committed lesson by lesson as I progress through the course.

## Topics Covered

- 3D scene basics (meshes, geometries, materials)
- Transforms, cameras, and lighting
- Animations with `requestAnimationFrame` and GSAP
- Shaders and WebGL fundamentals
- Interactive 3D web experiences

---

## Structure

Each folder is a standalone Vite project. To run any exercise:

```bash
cd 03-animations
npm install
npm run dev
```

## Stack
- Three.js
- GSAP
- Vite
